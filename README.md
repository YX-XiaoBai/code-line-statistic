# code-line-statistic

> A code line statistic tool base on tkinter✨💫.

![Python](https://img.shields.io/badge/-Python-333333?style=flat&logo=python)

## Usage

```bash
# 安装pipenv
pip3 install pipenv
# 安装项目依赖
pipenv install
# 运行项目
python3 src/main.py
```
